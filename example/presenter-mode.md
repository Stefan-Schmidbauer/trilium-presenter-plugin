## Three Ways to Present

::: {.columns}
::: {.column}
### Present

Fullscreen presentation in a new window.

Keyboard, mouse, and touch navigation with progress bar.
:::
::: {.column}
### Presenter Mode

Speaker view with:

- Current slide and notes
- Clickable slide list
- Synced with presentation window via BroadcastChannel
:::
::: {.column}
### Handout (PDF)

Print-optimized view — one page per slide.

Auto-opens the print dialog for PDF export.
:::
:::
