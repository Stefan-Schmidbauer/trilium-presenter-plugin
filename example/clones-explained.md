## How Clones Work

::: {.columns}
::: {.column}
### Clone = Reference

- Shares content with the original
- Edit once → updates **everywhere**
- Fix a typo in Master → fixed in all presentations

**Use for:** Shared topic slides
:::
::: {.column}
### Unique = Independent

- Exists only in this presentation
- Changes don't affect other sets

**Use for:** Title, Agenda, Contact slides
:::
:::

---

**How to clone:** Right-click a Master slide → **Clone to...** → select your Set. Or drag with **Ctrl** held.

::: {.notes}
Trilium shows a clone indicator on cloned notes. The presenter doesn't care whether a note is a clone or not — it just reads the content.
:::
